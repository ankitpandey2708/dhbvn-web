# Subscribe to Telegram Notifications

Get real-time power outage alerts delivered to Telegram when outages start or are restored in a chosen Haryana district.

## How to Subscribe

1. Open Telegram and search for **@dhbvn_bot** or visit [t.me/dhbvn_bot](https://t.me/dhbvn_bot)
2. Start a chat and send `/start`
3. Choose your district from the interactive menu
4. Done — you'll receive instant outage alerts

## Bot Commands

| Command    | Description                                           |
|------------|-------------------------------------------------------|
| `/start`   | Begin setup — choose a district to subscribe to       |
| `/status`  | Check your current subscription and recent outages    |
| `/change`  | Switch to a different district                        |
| `/stop`    | Unsubscribe from all alerts                           |
| `/help`    | Show available commands and instructions              |

## Features

- **Real-time alerts** when a new outage is reported
- **Restoration notifications** when power is back
- **Smart deduplication** — only notified of actual changes
- **No limits** — free for all users
